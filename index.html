HTML (англ. HyperText Markup Language - мова розмітки гіпертексту) - стандартизована мова розмітки документв для перегляду вебсторінок у браузері. Браузери отримують HTML документ від сервера за протоколами HTTP/HTTPS або відкривають з локального диска, далі інтерпретують код в інтерфейс, який відображатиметьс€ на екрані монітора.
